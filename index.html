Shortcuts
Use below combinations for shortcuts
  • CTRL + B for bolding text
  • CTRL + I for italizing text
  • CTRL + E for centering text
  • CTRL + SHIFT + U for underlining text
  • CTRL + SHIFT + R for aligning text to the right
  • CTRL + SHIFT + J for justifying text
  • CTRL + SHIFT + L for inserting link
